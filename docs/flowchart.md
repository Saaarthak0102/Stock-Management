# Stock Management System Flow

## High-Level Flow

```
Start
  |
  v
Display Banner
  |
  v
Login (max 3 attempts)
  |
  v
Main Menu
  |-- Stock Management
  |-- Supplier Management
  |-- Sales and Billing
  |-- Reports and Analytics
  |-- Admin Settings
  |
  v
Exit
```

## Data Layer

```
stock.dat     (products)
suppliers.dat (suppliers)
sales.dat     (sales)
users.dat     (users)
logs.dat      (activity)
```
